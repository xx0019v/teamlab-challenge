「Quiet Guide for Immersive Space」というコンセプトのもと、
展示作品への没入を邪魔しない、静かで直感的なガイドUIを目指しました。

課題1では、見た目の再現に留まらず、開発チームや他のデザイナーが
触ったときに迷わないよう、以下を意識しました。

・Auto Layout を前提とした柔軟な構造
・Color / Text Style / Component の徹底
・8ptグリッドと一貫した余白ルール
・暗所での視認性を考慮したコントラストと文字サイズ
・命名規則（英語snake_case）の統一

課題2では、展示会場の「暗所・回遊・身体的没入」という特徴を踏まえ、
低輝度・少情報・振動中心で案内する「Quiet Guide Mode」を提案しています。
アプリが体験の主役になるのではなく、作品体験を背景から支える存在に
なることを意識しました。

不足している点や、より深く伺いたい点がございましたら、
お気軽にお知らせください。
