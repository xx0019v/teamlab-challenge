## 提出手順

1. `Figma作業手順.md` の順で Figma を組む（最低限：Design System + Components + Home + Artwork Detail + Quiet Mode）。
2. Figma の共有リンクを「Anyone with link / Can view」で取得。
3. `補足資料.md` を Google Docs に貼り付け → PDFエクスポート。
4. `提出メール.md` の `[Figma URLをここに貼り付け]` を実URLに差し替え。
5. PDFを添付して送信。

## 制作意図（30秒で説明する版）

「Quiet Guide for Immersive Space」というコンセプトで、
**作品体験を邪魔しないUI**を目指しました。

- 暗所での視認性を優先し、本文14pt以上・コントラスト4.5:1以上を維持
- すべて Token・Component・Auto Layout で構造化し、開発時の再現コストを下げた
- 課題2では、展示会場専用の低輝度・少情報モード「Quiet Guide Mode」を提案し、
  歩きながらでも作品に集中できる体験を提案しています。
