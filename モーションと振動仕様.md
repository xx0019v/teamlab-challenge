# Motion & Haptics Specification

提出のもう一歩上を狙うために、動きと振動を仕様として書き出しておく。
Figma 上では Prototype として一部実装、残りは仕様シートとして提示する。

---

## 1. Easing & Duration トークン

| Token | Value | Use |
|---|---|---|
| `motion/duration/instant` | 80ms | Press feedback / Toggle |
| `motion/duration/quick` | 180ms | Hover / Small state change |
| `motion/duration/standard` | 240ms | Card 展開 / Sheet open |
| `motion/duration/slow` | 360ms | Page transition / Hero crossfade |
| `motion/easing/standard` | cubic-bezier(0.2, 0, 0, 1) | 既定 |
| `motion/easing/decel` | cubic-bezier(0, 0, 0, 1) | Enter |
| `motion/easing/accel` | cubic-bezier(0.3, 0, 1, 1) | Exit |
| `motion/easing/spring` | spring(damping 0.7) | Bottom Sheet / Snap |

ルール：
- Exit は Enter の 60–70% の時間
- prefers-reduced-motion が ON のときは Easing/standard のみ、duration 半減
- 同時アニメは2要素まで

## 2. 主な画面遷移

| From → To | 動き | Duration |
|---|---|---|
| Home → Area Detail | Card がフルブリードに拡張、Topbar が glass blur で滑り込む | 360ms slow + decel |
| Area → Artwork Detail | Hero 画像が shared element として遷移 | 360ms |
| Artwork Detail → Quiet Mode | 周囲を真黒に fade out、矢印が中央に拡大 | 400ms (例外) |
| Map sheet peek → expand | spring | 240ms |
| Audio Player play press | Play button が 1.05 → 1.0、progress bar に Water Blue glow が点灯 | 200ms |
| Quiet Mode arrow | 2拍 pulse (opacity 0.7 ↔ 0.85) | 1.6s loop |

## 3. Haptic 設計

| トリガ | パターン | 強度 | 用途 |
|---|---|---|---|
| Quiet Mode 入場 | Soft long (200ms) | 弱 | 「いまから静かになります」を通知 |
| 次の作品 28m 圏内 | Light single | 弱 | 接近 |
| 作品の前 5m | Light double | 中 | 到着 |
| 曲がる | Two short ticks | 弱 | 方向転換 |
| 混雑回避ルート変更 | Triple soft | 中 | 注意 |
| Quiet Mode 解除 | Soft long (200ms) | 弱 | 「通常に戻ります」 |
| エラー / Permission denied | Heavy single | 強 | 注意喚起 |

Haptic は 振動だけで意味が完結しないようにし、必ず同時に画面側で短いラベルかアイコンを 1.2 秒だけ提示する（聴覚障害のあるユーザーへの配慮）。

---

## 4. Audio Fade ルール（Quiet Mode と相性）

- ナレーション開始時：0 → 100% を 400ms で fade in
- 通常移動中：50% volume
- Quiet Mode：ナレーションは自動再生しない（ユーザータップ時のみ）
- 次の作品圏内に入ったとき：振動のみ、音は鳴らさない

---

## 5. Reduced Motion 時の代替

| 通常 | Reduced |
|---|---|
| Shared element transition | Crossfade only |
| Quiet Mode arrow pulse | Static |
| Splash の粒子 | Static dots |
| Bottom Sheet spring | Linear 180ms |
| Card hover scale | Background color change のみ |

すべて Settings → Accessibility → Reduce Motion で発動。
