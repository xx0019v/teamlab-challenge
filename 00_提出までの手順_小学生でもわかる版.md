# 🚀 提出までの手順（小学生でもわかる版）

提出物は **2つだけ**：
1. **Figmaファイルのリンク**（デザイン）
2. **PDFファイル**（補足資料）

それを **メールに貼って・添付して・返信** したら終わり。

⏰ **期限：今日 6/15(月) 10:00AM**

---

# STEP 1 — Figmaで最低限のデザインを作る（30分）

## 1-1. Figmaを開く

提供されたFigmaファイルを開く。
[https://www.figma.com/design/7RPUvieXKKEQitOAtLtFKr/teamlab-App---Designer-Entry-Challenge](https://www.figma.com/design/7RPUvieXKKEQitOAtLtFKr/teamlab-App---Designer-Entry-Challenge)

## 1-2. 自分のファイルにコピーする

1. 右上の **▼マーク** → **「Duplicate to your drafts」** をクリック
2. 自分のドラフトに複製されたファイルが開く

## 1-3. ページを作る

左のサイドバーで「+」を押して、以下のページを作る：

```
00_Cover
01_Design System
02_Components
03_Screens
04_UX Proposal
```

## 1-4. デザインシステムを登録する（最重要・15分）

`DESIGN_SYSTEM.md` を開いて、以下を Figma に登録する。

### A. カラー（Color Styles）

`02_Design System` ページで、四角を描いて、右パネルの **Fill → Style → +** で名前を付けて保存。

最低この5個だけ作る：

| 名前 | 値 |
|---|---|
| `Base/Black` | #050609 |
| `Surface/Card` | #181B22 |
| `Text/Primary` | #FFFFFF |
| `Text/Secondary` | rgba(255,255,255,0.72) |
| `Accent/Water Blue` | #69D9FF |

### B. テキストスタイル（Text Styles）

テキストを描いて、右パネルの **Text → Style → +** で名前を付けて保存。

最低この4個だけ作る：

| 名前 | 設定 |
|---|---|
| `Title/Large` | 24px / Bold / Line 1.3 |
| `Title/Medium` | 20px / Semibold / Line 1.3 |
| `Body/Large` | 16px / Regular / Line 1.6 |
| `Caption` | 12px / Regular / Line 1.4 |

> 💡 フォントは `Inter` か `Noto Sans JP` を選ぶ。

## 1-5. 画面を3つだけ作る（10分）

時間が無いので **Home / Artwork Detail / Quiet Mode の3画面だけ** 作る。

`03_Screens` ページで **Fキー** を押して、**iPhone 15 (393×852)** のフレームを3つ並べる。

### Home画面の作り方
1. 背景を `Base/Black` で塗る
2. 上に「teamLab Borderless」とテキスト（中央寄せ）
3. 真ん中に四角3つ縦に並べる（角丸18、高さ150ずつ）
4. それぞれに違う色のグラデーションを入れる
5. 下に浮かぶカード（角丸18、`Surface/Card`）を置いて「YOU ARE HERE / Borderless World」と書く

→ `提出パッケージ_完全版.md §2-2` に座標まで書いてある通りに置けばOK

### Artwork Detail画面
1. 背景 `Base/Black`
2. 上半分（300pt）に作品ビジュアル（青系グラデでOK）
3. その下に作品タイトル
4. その下に説明文
5. 一番下に [Quiet Mode] ボタン（`Accent/Water Blue`、角丸999、高さ44）

### Quiet Mode画面
1. 背景 **真っ黒 #000**
2. 中央に大きな矢印（96pt）
3. その下に「28m」（32pt 太字）
4. その下に作品名（小さく）
5. 下に「Swipe to exit」（とても小さく）

## 1-6. 共有リンクを取る

1. 右上の **「Share」** ボタンを押す
2. **「Anyone with the link」** に変更
3. **「Can view」** を選ぶ
4. **「Copy link」** をクリック

📋 これでFigmaのリンクがクリップボードに入った。

# STEP 2 — 補足PDFを作る（10分）

## 2-1. Google Docs を開く

[https://docs.google.com](https://docs.google.com) で **空白のドキュメント** を新規作成。

## 2-2. 中身を貼る

`補足資料.md` の中身を全部コピー → Google Docs に貼り付け。

## 2-3. 体裁を整える

- タイトル「teamLab Guide App UI/UX Redesign」を大きく
- 見出し（Page 1〜8）を「見出し1」スタイルに
- 本文は普通のまま

## 2-4. PDFで書き出し

1. メニュー **ファイル** → **ダウンロード** → **PDFドキュメント (.pdf)**
2. ダウンロードフォルダに保存される
3. ファイル名を **`teamLab補足資料_アヴェンダニョシンタロウ.pdf`** にリネーム

✅ これでPDFが出来た。

---

# STEP 4 — PDFを添付する

メール画面の下にある **📎クリップマーク**（添付）をクリック。
ダウンロードフォルダから `teamLab補足資料_アヴェンダニョシンタロウ.pdf` を選ぶ。

添付されたら、メール下部にPDFファイル名が表示される。

---

# STEP 5 — 送信前チェック → 送信！

送信ボタンを押す前に、次の **5つ** を確認：

- [ ] 宛先が **チームラボ** になっている（lab-talentio もCCに入ったまま）
- [ ] 件名が **「Re: [チームラボ]選考結果のご連絡[インターンシップ]」** のまま
- [ ] 本文の **`[ここにFigmaのリンクを貼る]`** が実際のリンクに置き換わっている
- [ ] PDFが **添付** されている
- [ ] Figmaのリンクを別タブで開いて、**ログアウト状態でも見られる** か確認（重要！）

全部OKなら、**送信** ボタンを押す。

🎉 **提出完了！**

# よくあるトラブル

| 困った | 解決 |
|---|---|
| Figmaリンクが「権限がない」と出る | Share設定で **「Anyone with the link」** にする |
| PDFが大きすぎる（25MB超） | Google Docsで保存し直すか、画像を減らす |
| 時間が足りない | Quiet Mode 画面だけでも出す。何も出さないより100倍マシ |
| 補足資料が間に合わない | 補足資料は **任意** なので、Figmaだけでも提出してOK |

# 時間配分（残り時間別）

## あと2時間ある場合
- 1時間 Figma（5画面）
- 30分 補足資料
- 30分 見直し＋メール

## あと1時間しかない場合
- 40分 Figma（3画面 + デザインシステム）
- 15分 補足資料（PDF化だけ）
- 5分 メール送信

## あと30分しかない場合
- 20分 Figma（Home + Quiet Mode の2枚 + 色4個）
- 0分 補足資料はスキップ
- 10分 メール送信

⚠️ どうしても間に合わなそうなら、メールに **「学業の都合で時間の確保が難しいため、提出期限を延長いただけますでしょうか」** と書いて返信すれば、最初のメールにも「別途相談可能」と書いてある。
