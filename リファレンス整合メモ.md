# リファレンス整合メモ（公式アプリ実物に基づく更新）

公式 [team-lab.com/guideapp/](https://www.team-lab.com/guideapp/) のスクリーンショットから観察した実構造と、提案版での扱いをまとめます。

---

## 観察した「公式アプリの実構造」

### Home
- **Topbar**: ☰ Hamburger(左) / "teamLab Borderless" 中央 / 📍 Location アイコン(右)
- **画面本体**: 全幅のエリア画像カードを**縦に積む**
  - "teamLab Borderless"
  - "Borderless World"
  - "Athletics Forest"
  - タイトルはカード中央にやや透ける形で配置
- **下部固定**: Floating な "Area where you are / Borderless World" カード（thumbnail + 2行ラベル）
- **Bottom Tab Navigation は存在しない**

### Artwork Detail
- **Topbar**: ‹ Back(左) / "Art work" 中央 / ☰ メニュー(右)
- **Hero 画像**: 上から差し込まれる作品ビジュアル
- **タイトル**: 2行に渡る長い英文タイトル
- **メタ**: "teamLab, 2018, Interactive Digital Installation"
- **本文**: 英語の説明文（長文）
- **音声プレーヤー**: ⏮ ▶ ⏭ + プログレスバー
- **下フッター**: 作品タイトル繰り返し

---

## 提案版での扱い

### 「再現」として残したもの
- Hamburger + 中央タイトル + 右アイコン の Topbar 構造
- エリアカードを縦に積む Home の構造
- 浮動 "You are here" カード
- 音声ガイドの存在
- 中央寄せ気味の白文字タイポグラフィ

### 「アップグレード」として加えたもの
| 要素 | Before（実物） | After（提案） | 理由 |
|---|---|---|---|
| Topbar | 単純な並び | Glass blur 20 + 細い線アイコン | 暗所での視認性と質感統一 |
| AreaHero Card | タイトルが画像に重なるだけ | eyebrow（FLORAL/YOU ARE HERE/ACTIVE）+ work count meta | 階層化と情報の歩留まり |
| Current Card | 単一の文字 | Water Blue accent eyebrow + chevron + 3 artworks nearby | アクションへの導線を明示 |
| Audio Player | 画面下にぶら下げ | 独立 Component（glass card / Water Blue progress / 44 round play） | 操作領域の確保と階層化 |
| Detail下部 | 何もなし | [Nearby] / [Quiet Mode] の2択 sticky | UX提案への自然な動線 |
| 新規 | — | Quiet Mode 画面 + Dark Map + Route | 課題2のUX提案として |

## 評価軸への対応（自分用メモ）

| 評価軸 | 対応 |
|---|---|
| Auto Layout / 構造 | すべての Card / Player / Topbar を AutoLayout で組み、画像差し替えで自動再構成 |
| 再利用性 | AreaHero / AudioPlayer / FloatingCard をプロジェクト全体で1個ずつ作り Variant 切替 |
| 暗所での視認性 | 本文14pt以上、白透明度4階層、Water Blue は1色のみ、コントラスト 4.5:1 以上 |
| 没入感を邪魔しないUI | 装飾を減らし、glass blur と eyebrow ラベルで階層を伝える |
| チームラボらしさ | Water Blue の glow を最小限の場所だけに残し、過剰な装飾を避ける |

---

## 必ず言及すべき「再現と提案の境界」（面接対策）

> 「実物のHomeはエリアカードを縦に積み、Bottom Navを持たず、現在地カードで居場所を伝える構造でした。この構造は来場者が**どこにいるか**を最初に伝える設計として理にかなっていると考え、変えていません。
> 一方で、暗所での視認性と次の行動への導線が弱いと感じたため、eyebrow ラベル・glass blur・Water Blue の最小限のアクセントを足し、Detail には音声ガイドの上に Quiet Mode への入り口を追加しました。」
